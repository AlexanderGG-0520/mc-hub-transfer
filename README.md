# mc-hub-transfer

mc-hub-transfer is a server-side Fabric mod that adds `/hub`.

## Support

- Loader: Fabric
- Minecraft: 26.1.2
- Mod version: 0.1.0

## Usage

```text
/hub
```

Install the mod and Fabric API on each non-hub server. Players who run `/hub`
are transferred to `play.alec-ofc.com:25565`.
